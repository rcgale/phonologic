## Working *on* PhonoLogic Viewer

### Installing tools
- [Node.js](https://nodejs.org) (tested on v16)
- Cargo
- `cargo install wasm-pack`

### Setting up project

```shell
cd phonologic-js
npm install
npm run build
```

### Running a local instance

```shell
npm run start
```
